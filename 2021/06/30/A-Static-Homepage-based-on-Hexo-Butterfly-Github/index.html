A Static Homepage based on Hexo + Butterfly + Github

Created2021-06-30|Updated2021-06-30|Web
|Word count:274|Reading time:1min|Post View:

Configuration

  • os: Windows_NT 10.0.19042 win32 x64
  • node: 14.16.1
  • hexo: 5.4.0
  • butterfly: 3.7.8

Front-matter

Front-matter is the top area of each markdown file, used to designate various variables of certain files.

  • Page Front-matter is used for Page configurations
  • Post Front-matter is used for Post configurations

Tags Page

Remember to add type: tags into the Front-matter of Tages Page.

Categories Page

Remember to add type: categories into the Front-matter of Categories Page.

Link Page

Remember to add type: link into the Front-matter of Link Page. So far I don’t have any link pages from any friends yet, therefore I leave it blank, to be added later on.

Gallery

Just an ordinary page, so use hexo new page xxxx to create one page.

Markdown grammar is as follows:

https://fineq.cn/gallery is an example how it looks.

Sound Rise II SFQ-16

Sound Rise II是一款理想的床头数字闹钟蓝牙音箱，适合讲究现代生活风格的家庭卧室或者酒店客房。

Sound Rise第一代产品，自推出之后就在国外获奖无数，广受市场欢迎。Sound Rise II，在保留了第一代产品Sound Rise音质的基础上，又追加了一些额外的新功能，例如更大的屏幕尺寸，更多的充电接口选项，以及用于舒缓情绪和改善睡眠的自然界声响。静躺床上小憩，可以欣享自己最喜欢的音乐，自然而愉悦。现代化的风格，小巧化的设计，摆放在床头柜上几乎不占空间。

声音表现

  • 专门定制的单声道喇叭单元，全音域自然平衡，人声清晰
  • 无源辐射器，增强低频的重建，低音更深沉
  • 5瓦的音频功率输出，不破音，无失真
    （*酒店版本调整为3W）

功能特点

  • 使用场景：通过滑动开关选择“酒店”或者“家庭”
  • USB充电接口 x 3个
    • 顶部，2个：支持DC 5V 2.4A快充，最大输出功率12瓦 + 12瓦
    • 背部，1个：符合QC3.0标准，根据插入的USB配件之需求功率而定
      • 既支持DC 5V 2.4A快充，最大输出功率12瓦
      • 也支持DC 9V 2A或者DC 12V 1.5A，最大输出功率18瓦
  • 自然界声响：鸟鸣，森林，海浪，雨滴，河流，风吹 - 可使人放松，保持专注，改善睡眠
  • 旋钮控制：轻松调节音量，选台，或者设定闹钟
  • 内置纽扣电池和内存芯片，可以保证闹钟设置和用户设置不会掉电丢失
  • 无线蓝牙：轻松从已经配对成功的移动设备上播放喜爱的音乐
  • FM收音机：10个预设电台（仅限“家庭”版本）
  • 全功能闹钟：贪睡功能，闹铃音量逐渐升高
  • 可选择闹铃：蓝牙音乐 / FM收音机 / 内置闹铃
  • 时间显示模式：12小时 / 24小时
  • 睡后定时自动关机：5种可选时间间隔，15 / 30 / 60 / 90 / 120分钟
  • 屏幕亮度调节：4档亮度可选
  • 夏令时间设定：通过滑动开关可轻松地拨快 / 拨慢1小时
  • 3.5mm模拟音频输入接口
  • 防盗安全挂绳 + 接口软胶盖 （仅限“酒店”版本）
  • 接触安全：防菌防微生物塑胶按键
  • 金属喇叭网：方便日常的清洁，维护和消毒
  • 防火安全：机身采用防火等级UL94V0的塑胶外壳

产品尺寸

  • 宽 5.2” / 132mm

  • 高 5.9” / 149mm

  • 深 3.5” / 90mm

床头闹钟

首次用户设置

提示：酒店和家庭两个版本的首次使用设置过程二者是相同的。

用户操作过程如下：

  1. 插入电源适配器，屏幕会闪烁和显示 —:— 提示设置

  2. 拨动机身背部的滑动开关，以选择酒店或者家庭使用环境（HOTEL代表酒店，HOME代表家庭）

  3. 长按SET键3秒以进入设置模式

    设置日历

    • 代表年份的2021会显示和闪烁，转动TUNE/SET旋钮来选择和设定想要的年份，按ENTER确认。
    • 代表月份的12会显示和闪烁，转动TUNE/SET旋钮来选择和设定想要的月份，按ENTER确认。
    • 按照同样的步骤设定日期。

    设置时间和时间格式

    • 默认时间会显示为早上9点（9AM）。
    • 小时会显示和闪烁，转动TUNE/SET旋钮来选择和设定想要的时数，按ENTER确认。
    • 按照同样的步骤设定分钟。
    • 12会显示和闪烁，转动TUNE/SET旋钮或者按”上一首“、”下一首“（ << / >> ） 来选择和设定想要的时间格式，12时制或24时制，按ENTER确认。
    • 首次时间设置完毕。

  4. 设置完成以后，LCD屏幕会显示时间，日期和星期信息。闹钟时间默认为早上6点（6AM），闹铃状态为关闭，屏幕上不显示闹铃图标。

  5. 显示年份：在待机状态下，短按PALY/PAUSE键，屏幕会显示年份信息，持续3秒后转为显示当前时钟时间。

  6. 在首次时间设置完成以后，音箱会进入待机状态。这时按STANDBY键，可以开机进入播放模式。

  7. 任何时候，都可以通过按住SET键3秒，并重复上面的步骤来更改年份、时间和时间格式（12小时 / 24小时）。

  8. 在酒店客房内使用时，请在螺丝口位置安装防盗绳和安全卡扣。

何为夏令时间

夏令时，表示为了节约能源，人为规定时间的意思。也叫夏时制，夏时令（Daylight Saving Time：DST），又称“日光节约时制”和“夏令时间”，在这一制度实行期间所采用的统一时间称为“夏令时间”。一般在天亮早的夏季人为将时间调快一小时，可以使人早起早睡，减少照明量，以充分利用光照资源，从而节约照明用电。各个采纳夏时制的国家具体规定不同。全世界有近110个国家每年要实行夏令时。

美国地区

大部分地区，从每年三月份的第二个周日凌晨2点，会自动将时间加快1个小时，变成凌晨3点；从十一月份的第一个周日凌晨2点，再自动将时间减慢1个小时，变成凌晨1点。

以2021年为例：

在当地时间要到达2021年3月14日，星期日，02:00:00 ，时钟会自动加快1个小时，变成03:00:00

这样，3月14日的日升和日落会比前一日都晚了大约1个小时，傍晚的白天时间会更多一些。

在当地时间要到达2021年11月7日，星期日，02:00:00 ，时钟会自动减慢1个小时，变成01:00:00

这样，11月7日的日升和日落会比前一日都早了大约1个小时，早晨的白天时间会更多一些。

中国地区

1986年4月，中国中央有关部门发出“在全国范围内实行夏时制的通知”，具体做法是：每年从四月中旬第一个星期日的凌晨2时整（北京时间），将时钟拨快一小时，即将表针由2时拨至3时，夏令时开始；到九月中旬第一个星期日的凌晨2时整（北京夏令时），再将时钟拨回一小时，即将表针由2时拨至1时，夏令时结束。从1986年到1991年的六个年度，除1986年因是实行夏时制的第一年，从5月4日开始到9月14日结束外，其它年份均按规定的时段施行。在夏令时开始和结束前几天，新闻媒体均刊登有关部门的通告。1992年起，夏令时暂停实行。

夏令时间设定

Sound Rise II在产品内部有一个内置的万年历，当年份、月份和日期在设置成功以后，Sound Rise II能够自动计算和显示当前日期是周几，用户并不需要再额外手动地进行输入和设置。这个内置的万年历也同样可以支持AutoDST（自动夏令时）的功能。在Sound Rise II机身背部有一个滑动开关，并设置了三个档位

  • A：美国地区用户，可以调到这个档位，闹钟将根据内置万年历适时地自动加快或者减慢1个小时。这时屏幕会显示“DST”的图标。
  • 0：非美国地区用户，可以手动调到这个档位，以取消夏令时间。因为中国地区早已经取消夏令时间，用户可以选择将滑动开关拨动到此位置。
  • 1：非美国地区用户，可以手动调到这个档位，以加快1个小时。这时屏幕会显示“DST”的图标。

屏幕亮度调节

机顶的DIMMER按键可以用来调节屏幕的亮度，共有为6个级别

级别 亮度
5 100%（最高亮度）
4 75%
3 50%
2 25%
1 5%
0 0（背光熄灭）
  • 按下DIMMER按键可以依次在6个亮度级别间切换，5 $\rightarrow$ 4 $\rightarrow$ 3 $\rightarrow$ 2 $\rightarrow$ 1 $\rightarrow$ 0 $\rightarrow$ 5
  • 如果用户选择把背光的亮度级别设置得比较低，比如设置为0级和1级时
    • 在待机状态，当遇到有用户操作（比如按键操作或者旋钮操作）时，屏幕背光会自动点亮并保持3级亮度，持续3秒，以方便用户操作
    • 在待机状态，当遇到有闹钟启动发出闹铃时，屏幕背光会自动点亮并保持3级亮度，直到有用户介入（比如按Snooze贪睡或者关闭闹钟）

蓝牙音箱

配对操作

  1. 短按SOURCE键，选择BLUETOOTH作为音源，此时，在未有蓝牙连接或者蓝牙连接已断开的状态下，屏幕上的蓝牙图标将会慢闪。
  2. 要开始配对，短按PAIR键，这时屏幕上的蓝牙图标将会快闪。注意，在此操作过程中如果需要取消配对，可以按住PAIR键2秒，蓝牙图标将变为慢闪。
  3. 待蓝牙连接成功以后，屏幕上的蓝牙图标将变为固定显示，不再闪烁。这时候就可以使用蓝牙进行无线传输了。
  4. 跟另外一台设备配对：在蓝牙播放状态下，如果长按PAIR键2秒就可以使当前的蓝牙连接断开，重新开始配对过程。此时，蓝牙图标将会变为快闪并准备跟另外一台设备配对。
  5. 自动配对：如果音箱曾经与某台设备配对过，当音箱的音源输入切换到BLUETOOTH以后，音箱为自动跟这台设备进行蓝牙配对。
  6. 在蓝牙播放过程当中，如果短按PAIR键，此时按键应该没有任何作用。

重要提示

  • 系统重置后，第一次使用音箱，此时应该没有保存的蓝牙设备了。这种状态下，寻找保存设备和自动配对的过程会被跳过，直接进入到正常的设备配对流程，以节省时间。
  • 如果蓝牙音源被设定为闹铃提示音，当闹钟触发时，音箱会寻找所有保存过的蓝牙设备，然后尝试自动配对和播放。

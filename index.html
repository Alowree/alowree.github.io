Soundfreaq旗舰蓝牙音箱Sound Stack需要煲机吗？

发表于 2021-04-30 更新于 2021-05-07 分类于 声学 Valine：
本文字数： 3.7k 阅读时长 ≈ 3 分钟

背景

有一位来自福建做茶业生意的老铁，关注SFQ-03有一段时间了，也添加了微信好友。但是，由于经营生意经常在外奔波，无法收货，故而一直没下单，直到上周人返福建。下单后的第三天，这位老兄收到了音箱。开箱试用了不一会儿，很是满意和欣喜，特意地拍照、拍视频、写评论，还给我发来了截图。

不得不说，这位兄台写的评论，拍的视频，简直堪比专业刷单的枪手。这台Sound Stack的彩花版，也就是Novogratz的定制版，跟他家里橱柜上的一只景德镇手绘粉彩将军罐简直是绝配，音箱与摆罐，简直了就是门当户对的一妆婚配。

这款旗舰蓝牙音箱使用了两天之后，这位兄台在微信上发信息给我，提出了一个问题：这款音箱大约需要多长的煲机时间？

煲机这个术语，在音响行业肯定会经常听到人们提起，但是个人来讲对它没有怎么过多的关注。有一个主要的原因，我认为自己的耳朵就是一对普通人的“肉耳朵”（也有人称“木耳朵”），不是专业声学工程师们通常所具有的“金耳朵”，可以分辨出哪怕是最细微的声音差别。听过最夸张的一个故事，公司之前的技术顾问Kitazawa就是这样一个声学专家，喇叭供应商在某一批喇叭单元的生产中仅是更换了一条引线，Kitazawa都可以听出喇叭更换引线前后的声音差别。

作为一个音响行业的从业者，木头耳朵也就罢了，如果又是木头脑袋那就糗得大了。

现在用户提出了一个专业的声学问题，为使我看起来像是一个具有行业知识的专业人士，我很想若有其事的告诉他，你可以使用某某类型的音乐，大约煲个百把小时，然后就可以“低频汹涌澎湃，人声细腻悦耳，高频柔顺，总之一个字就是通透”，blah blah blah。然而，这不并是我真实的想法，如果客人针对这一话题打破沙锅问到底的话，我没有办法提供更多的信息并且自圆其说。

我决定实事求是，说出我自己真实的看法。如果连我这样的行业人士都不觉得煲机真正有用，或许这也正是真相所在，至少是对于我所熟悉的产品类型，应是如此。

文件，就是计算机中由操作系统管理的具有名字的存储区域。内置open()函数会创建一个Python文件对象，可以作为计算机上的一个文件链接。与我们目前见过的类型相比，文件对象多少有点不寻常，它们不是数字、序列、也不是映射。

文件操作

在 Python 中，如果想要操作文件，首先需要创建或者打开指定的文件，并创建一个文件对象，而这些工作可以通过内置的 open() 函数实现。

  • file_name：要创建或打开文件的文件名称，该名称要用引号（单引号或双引号都可以）括起来。需要注意的是，如果要打开的文件和当前执行的代码文件位于同一目录，则直接写文件名即可；否则，此参数需要指定打开文件所在的完整路径。

  • encoding：手动设定打开文件时所使用的编码格式，不同平台的 encoding 参数值也不同，以 Windows 为例，其默认为 cp936（实际上就是 GBK 编码）。

关于Soundfreaq

Soundfreaq是一个在欧洲和北美市场运营了10年的年轻音响品牌，在零售渠道和星级连锁酒店渠道都有相当的知名度和美誉度。2010年，Soundfreaq先是从北美市场开始起步，在线下渠道先后进入Target连锁超市和当地运营商们的数码3C渠道，在线上则有Shopify和亚马逊美国站。在欧洲市场，线下渠道跟Primark进行合作，线上渠道以英亚为主，德亚正在筹建当中。由于Soundfreaq安全可靠、品质稳定、音色自然，其产品也得以成功地进入到全球各大星级边锁酒店的客房，部分的合作伙伴包括丽兹卡顿，希尔顿，万豪，君悦…等一线品牌星级酒店。

关于声音的常见度量参数

发表于 2020-07-07 更新于 2021-04-30 分类于 信号处理 Valine：
本文字数： 7.9k 阅读时长 ≈ 7 分钟

声压，声强和声功率

声压 Sound Pressure

声压，$p$，就是大气压受到声波扰动后产生的压强变化，它相当于在标准的大气压强上叠加了一个声波扰动之后，所引起的压强变化。也就是说，空气中有声场时的压强 $p_s$ 与没有声场时的压强 $p_0$ 之差即为声压。

声压的单位是帕斯卡，$Pa$，声压的参考值是 $20\mu Pa$，这个数值表示人耳在 $4000Hz$ 处的平均可听阀值，换句话说，是人耳在 $4000Hz$ 处可被感知的平均最小声压波动值。

听阀下限：要注意，我们这里所说的是，年轻、健康的人耳其听觉的声压阀值下限大约是 $2\times 10^{-5} N/m^2$，或者说 $20\mu Pa$，低于这个阀值人耳是无法感知的。最低可听声压点大约位于 $4,000Hz$ ，这是一个由分子热运动所造成的物理极限。 国际上一般公认地将这个阀值下限定义为参考声压$p_{ref}$。

听阀上限：对正常的人耳来说，当声源产生的有效声压达到了$60Pa$级别的时候，就会感到疼痛。 为什么要使用有效声压？这是因为，测量的声压是变化的声压与静压强之差，声压的变化时正时负，其平均值为零，所以，平均声压不是一个有用的参量。而人耳对瞬时声压波动也没有响应，但对动态声压的均方根值（$RMS$）有响应，且平均响应时间间隔约为$35ms$ 。因此，声压测量的是有效声压$p_{rms}$ 。
